🌍 Travel Tourism Website

---

📋 Description

This repository contains a frontend Travel & Tourism website built for practice and learning. It showcases different destinations and basic travel-related pages.

---

📁 Pages Included

1. 🏠 Home Page
   Displays main travel destinations and overview

2. 📦 Packages Page
   Shows travel packages and details

3. 📄 Privacy Page
   Basic privacy information

4. 📜 Terms Page
   Terms and conditions of the website

---

🛠 Technologies Used

HTML5
CSS3
JavaScript

---

🚀 How to Run

Open the project folder
Run the "index.html" file in your browser
