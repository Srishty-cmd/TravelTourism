<h1>🌐 Personal Portfolio</h1>

📋 Description

This repository contains a full-stack portfolio website built to showcase projects, skills, and achievements. It includes a modern UI along with backend integration for handling contact form submissions.

---

📁 Sections Included

1. 🏠 Home Page
   Displays introduction and overview

2. 📂 Projects Page
   Shows featured projects with details

3. 📄 Resume Section
   Preview and download resume

4. 📬 Contact Page
   Allows users to send messages (stored in database)

---

🛠 Technologies Used

Frontend:
React.js
CSS

Backend:
Node.js
Express.js

Database:
MongoDB

---

🚀 How to Run

1. Open the project folder

2. Setup backend:

cd backend
npm install
npm run dev

3. Setup frontend:

cd frontend
npm install
npm run dev
